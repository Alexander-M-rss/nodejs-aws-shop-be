## Available Scripts

### `cdk:bootstrap`

Makes CDK bootstrap.

### `cdk:deploy`

Deploys lambda functions to AWS.

### `cdk:remove`

Removes Product service infrastructure from AWS.

### `test`

Runs unit tests.

### `lint`, `prettier`

Runs linting and formatting for all files in `src` folder.
